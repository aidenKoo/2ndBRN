# PLAN

## Phase 1 — Foundation (완료)
- 문서 구조/역할/게이트/스킬 골격 구축

## Phase 2 — Operationalization (진행 예정)
- CLI 명령(`/plan`, `/run`, `/check`, `/report`) 설계
- 자동 게이트 실행 스크립트 추가
- 리포트 자동 생성 포맷 고정

## Phase 3 — Evolution (예정)
- 스킬 메타데이터 집계(성공률/실패태그)
- 자동 승급/강등 정책 적용
- failure-patterns 기반 self-healing 루프 적용
