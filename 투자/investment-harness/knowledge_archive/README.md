# Knowledge Archive

투자 이론, 도서, 논문, 사례, 개념을 저장하고 이를 하네스 룰 개선에 활용하기 위한 지식 저장소이다.

## 목적
```txt
투자 지식 → 투자 원칙 → 검증 가능한 룰 → 하네스 반영
```

## 상태
- raw
- summarized
- interpreted
- rule_candidate
- tested
- adopted
- rejected
- archived

## 금지 사항
- 출처 없는 투자 격언을 룰로 반영 금지
- 단일 성공 사례를 보편 룰로 승격 금지
- 실패 사례 무시 금지
- 검증 없는 active rule 반영 금지
