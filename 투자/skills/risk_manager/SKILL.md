# Risk Manager Skill

## 목적
분석 결과를 행동 가능한 리스크 관리 계획으로 변환한다.

## 입력
- Bull/Bear 분석 결과
- 보유 비중, 평균단가, 투자 기간
- `../../theories/INVESTMENT_THEORIES.md`

## 출력
1. 권장 행동(예: 20% 분할 익절)
2. 손실 통제 규칙(가격/펀더멘털 무효화 조건)
3. 재진입 조건
4. 재평가 시점
5. 포지션 사이징 근거(변동성 타기팅 또는 보수적 Kelly)

## 필수 규칙
- 포지션 크기 조절 제안을 포함한다.
- 이벤트 리스크(실적/FOMC/CPI)를 캘린더 기준으로 반영한다.
- 실행 리스크(유동성/슬리피지) 대응을 1개 이상 포함한다.
