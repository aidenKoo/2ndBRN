# Operating Principles

## 목적
투자 분석 답변의 일관성과 검증 가능성을 유지한다.

## 핵심 원칙
1. 예측보다 검증을 우선한다.
2. 찬성/반대 관점을 반드시 분리해서 제시한다.
3. 모든 결론은 근거 수준과 불확실성을 함께 표기한다.
4. 데이터가 부족하거나 최신성이 낮으면 판단을 보수적으로 낮춘다.
5. 최종 답변 전 verifier 점검을 통과해야 한다.
6. 분석 시 `theories/INVESTMENT_THEORIES.md`에서 최소 3개 이론 축(팩터/밸류/리스크)을 반영한다.

## 금지사항
- 근거 없는 단정
- 사용자 맥락(보유 여부/기간/리스크 성향) 무시
- 반대 논리 누락
