# Earnings Reaction Playbook

## 목적
실적 발표 전후 단기 대응 판단을 일관되게 생성한다.

## 필수 입력
- ticker
- earnings_date (현지시간)
- actual vs consensus (EPS/Revenue/Guide)
- 포지션 정보

## 실행 절차
1. 실적 수치 서프라이즈/미스 분해
2. 가이던스 변화와 컨콜 코멘트 분리
3. 가격 갭 + 거래량 반응 해석
4. bull/bear/risk/verifier 실행
5. 이벤트 리스크 정책 반영(확신도 하향)

## 출력
- 실적 해석 요약(팩트/의견 분리)
- 24h/1w 시나리오
- 무효화 조건
- 재평가 시점
