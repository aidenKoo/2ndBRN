# Single Stock Analysis Playbook

## 목적
단일 종목의 매수/보유/익절/관망 판단 보조 절차를 정의한다.

## 호출 조건
- 특정 종목의 대응 전략 질문
- 보유 종목의 익절/손절 여부 질문

## 필수 입력
- ticker
- 보유 여부
- 평균단가(보유 시)
- 투자 기간
- 질문 목적

## 이론 참조 기준
- 분석 단계에서 `../theories/INVESTMENT_THEORIES.md`를 참조한다.
- 최소 반영 이론: 팩터(시장+1개 이상), 밸류에이션, 리스크 관리.

## 실행 절차
1. 사용자 포지션 맥락 확인
2. 이벤트/가격 흐름 분리
3. `bull_analyst` 실행
4. `bear_analyst` 실행
5. `risk_manager` 실행
6. `verifier`로 최종 점검
7. `../evals/theory_checklist.md`로 이론 커버리지 점검

## 최종 출력 필수 항목
- 현재 판단
- 상승 논리
- 하락 논리
- 핵심 리스크
- 행동 제안
- 무효화 조건
- 다음 체크포인트

## 실패 조건
- 반대 논리 누락
- 리스크 누락
- 사용자 맥락 미반영
- 이론 체크리스트 Medium 미만
